Automated E-Commerce Analytics System using Machine Learning

#Project Overview

This project is an end-to-end E-Commerce Analytics System that combines:

MySQL Database
Python Automation
Machine Learning
Real-Time Prediction
Power BI Dashboard

The system automatically reads sales data from MySQL, trains a machine learning model, predicts product prices, detects anomalies, generates alerts, stores predictions back into MySQL, and updates a Power BI dashboard.

#Features
Data Collection:
        Reads order data from MySQL
Machine Learning:
        Linear Regression model
        Predicts  future product prices
Real-Time Automation:
        Pipeline runs automatically every 10 seconds
Anomaly Detection:
        Detects unusual price spikes and drops
Alert System:
        Low Price Alert
        High Price Alert
        Normal Status
Dashboard Visualization:
        Product-wise sales analysis
        City-wise sales analysis
        Predicted price trends
        Alert monitoring


#Technologies Used:
Python
Pandas
Scikit-Learn
MySQL
Power BI
Schedule Library

#Workflow:
1.Order data stored in MySQL
2.Python loads data
3.ML model trains automatically
4.Price prediction generated
5.Alerts created
6.Results stored in MySQL
7.Power BI dashboard refreshes
